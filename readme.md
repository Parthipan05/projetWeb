## Description

StationFinder permet à l'utilisateur de :

- Sélectionner sa région sur une carte interactive
- Choisir son département et sa ville
- Consulter les prix des carburants en temps réel
- Trouver les stations les plus proches via géolocalisation IP
- Basculer entre mode jour et mode nuit (cookie)
- Consulter les statistiques de consultation

## Technologies utilisées

- HTML5, CSS3, PHP 8
- APIs : data.economie.gouv.fr, ipinfo.io, ip2location.io, ghibliapi.vercel.app
